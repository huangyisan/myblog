go语言基本类型以及之间的转换

Posted on 2020-06-14 |

go语言基本类型

说难不难，说容易也不容易，得把这些类型全部搞清楚。

  1. 数字类型

    • 整形

      • 有符号型

      | Type | Size | Range |
      | :———: | :—————– | ———————————– |
      | int8 | 8 bits | -27 ~ 27-1 |
      | int16 | 16 bits | -2 15 ~ 215-1 |
      | int32(rune) | 32 bits | -2 31 ~ 231-1 |
      | int64 | 64 bits | -2 63 ~ 263-1 |
      | int | 视当前操作系统而定 | 视当前操作系统而定 |

      * 无符号型

      | Type | Size | Range |
      | :———: | —————— | ——————– |
      | uint8(byte) | 8 bits | 0 ~ 28-1 |
      | uint16 | 16 bits | 0 ~ 216-1 |
      | uint32 | 32 bits | 0 ~ 232-1 |
      | uint64 | 64 bits | 0 ~ 264-1 |
      | uint | 视当前操作系统而定 | 视当前操作系统而定 |

      • byte和rune类型

        1. byte类型实则为unit8类型，他用来存放ASCII字符。

        2. rune类型实则为int32类型，他用来存放以utf-8编码的unicode字符类型。

        3. byte和rune都是用单引号引起来，比如’A’, ‘黄’。
        4. 如果没有指定类型，则默认为rune类型，比如var char = ‘A’, 此时char是rune类型。
        5. 如果要用byte类型，则需要声明的时候指定byte类型，比如var char byte = ‘A’，此时char是byte类型。
        6. 因为byte和rune类型，实际是整数，所以如果对于’A’的存放使用byte，则是存了65这个值。

    • 浮点型

      | Type | Size | 有效位 |
      | :—–: | —— | —— |
      | float32 | 32bits | 7位 |
      | float64 | 64bits | 15位 |

      • float32

        单精度，存储占用4个字节，也即4*8=32位，其中1位用来符号，8位用来指数，剩下的23位表示尾数。

      • float64

        双精度，存储占用8个字节，也即8*8=64位，其中1位用来符号，11位用来指数，剩下的52位表示尾数。

  1. 字符类型

    • 字符类型strings是一串byte，使用双引号，或者单反引号来表示，"hello", \hello``。
    • 双引号内的字符串，不可以跨行，且会受到转义字符的影响，比如\n,\t，如果想要不受影响，则使用单反引号。

  2. 布尔类型

    • bool类型只有false和true两种，且不可以像python那样存在隐式转换。

类型之间的转换

  1. go语言不存在类型之间的隐式转换，都需要进行显式转换。
  2. 转换可以从小范围往大范围转换，也可以大范围往小范围转换，但大范围转小范围往往会导致结果溢出。

数字类型之间的转换

只需要进行T(v)的方式即可

  1. int32转int64

  1. int32转float32

  2. float64转int64，会把小数部分掐掉

  1. 大范围转小范围容易产生结果溢出

跟字符串之转换

  1. 使用fmt.Sprintf方式进行转换, 该方法传入对象，以及对象所属类型，然后转换成string类型返回。
