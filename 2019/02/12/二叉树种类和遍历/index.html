дәҢеҸүж ‘з§Қзұ»е’ҢйҒҚеҺҶ

Posted on 2019-02-12 Edited on 2020-12-26 Disqus:

дәҢеҸүж ‘з§Қзұ»

дәҢеҸүж ‘з§Қзұ»е®ҡд№үжІЎжңүдёҖдёӘж ҮеҮҶеҢ–пјҢжүҖд»ҘдёҚеҗҢж–ҮжЎЈеҸҜиғҪжңүдёҚеҗҢи§ЈйҮҠгҖӮ

  1. rooted binary tree
  • еӯҳеңЁж №иҠӮзӮ№пјҢе№¶дё”жҜҸдёӘиҠӮзӮ№йғҪжңү2дёӘеӯҗиҠӮзӮ№гҖӮ
  1. full binary tree
  • жҜҸдёӘиҠӮзӮ№еҸӘжңү0дёӘжҲ–иҖ…2дёӘеӯҗиҠӮзӮ№гҖӮ
  1. complete binary tree
  • йҷӨеҺ»жңҖеҗҺдёҖзә§пјҢе…¶д»–зә§йғҪжҳҜfullзҠ¶жҖҒ(0дёӘжҲ–иҖ…2дёӘеӯҗиҠӮзӮ№)пјҢдё”жңҖеҗҺдёҖзә§зҡ„иҠӮзӮ№йғҪеңЁе·Ұдҫ§еӯҗж ‘дёҠгҖӮ
  1. perfect binary tree
  • еҶ…иҠӮзӮ№йғҪжңүдёӨдёӘеӯҗиҠӮзӮ№пјҢдё”жүҖжңүеӯҗеҸ¶йғҪеӨ„дәҺеҗҢдёҖзә§еҲ«гҖӮ
  1. infinite complete binary tree
  • every node has two children (and so the set of levels is countably infinite). The set of all nodes is countably infinite, but the set of all infinite paths from the root is uncountable, having the cardinality of the continuum
  • жҲ‘зҡ„зҗҶи§Је°ұжҳҜж— з©·е°Ҫзҡ„rooted binray tree
  1. balanced binary tree
  • е·ҰеҸіеӯҗж ‘й«ҳеәҰе·®дёҚи¶…иҝҮ1
  • дё”е·ҰеҸіеӯҗж ‘еҗ„иҮӘеқҮдёәе№іиЎЎдәҢеҸүж ‘
  1. degenerate tree
  • жҜҸдёӘиҠӮзӮ№еҸӘеҢ…еҗ«дёҖдёӘеӯҗиҠӮзӮ№гҖӮ

йҒҚеҺҶж–№ејҸ

еүҚеәҸйҒҚеҺҶ(ж·ұеәҰдјҳе…Ҳ)

* жҢҮе…Ҳи®ҝй—®ж №пјҢз„¶еҗҺи®ҝй—®еӯҗж ‘зҡ„йҒҚеҺҶж–№ејҸгҖӮ * йЎәеәҸ `F, B, A, D, C, E, G, I, H.`

е Ҷж Ҳд»Јз Ғе®һзҺ°:

и§ЈйҮҠпјҡ
д»ҺжңҖйЎ¶еұӮзҡ„ж №иҝӣе…ҘпјҢз„¶еҗҺжү“еҚ°еҮәиҝҷдёӘж №пјҢе№¶дё”еҺӢе…Ҙж ҲпјҢз„¶еҗҺиҺ·еҸ–иҝҷдёӘж №зҡ„е·ҰиҠӮзӮ№пјҢдҫқж¬ЎеҫӘзҺҜпјҢзӣҙеҲ°е·ҰиҠӮзӮ№ж— жі•иҺ·еҸ–еҲ°пјҢз„¶еҗҺеҮәж ҲпјҢиҝҷиҫ№еҮәж ҲеҮәжқҘзҡ„дёәжңҖеҗҺеҺӢе…Ҙзҡ„ж №(д№ҹе°ұжҳҜеә•еұӮзҡ„е·ҰиҠӮзӮ№)пјҢз„¶еҗҺејҖе§ӢйҒҚеҺҶе…¶еҸіиҫ№зҡ„еӯҗж ‘пјҢеҸіиҫ№еӯҗж ‘д№ҹжҳҜд»Һе·Ұиҫ№ејҖе§ӢпјҢд№ҹжҳҜеҺӢе…Ҙж ҲпјҢдҫқж¬ЎеҫӘзҺҜгҖӮ

дёӯеәҸйҒҚеҺҶ(ж·ұеәҰдјҳе…Ҳ)

* жҢҮе…Ҳи®ҝй—®е·ҰпјҲеҸіпјүеӯҗж ‘пјҢз„¶еҗҺи®ҝй—®ж №пјҢжңҖеҗҺи®ҝй—®еҸіпјҲе·Ұпјүеӯҗж ‘зҡ„йҒҚеҺҶж–№ејҸгҖӮ * йЎәеәҸ `A, B, C, D, E, F, G, H, I.`

е Ҷж Ҳд»Јз Ғе®һзҺ°:

и§ЈйҮҠпјҡеӣ дёәжҳҜе·ҰеҸідёӯзҡ„ж–№ејҸпјҢжүҖд»ҘrootжҳҜжңҖиҝҹжү“еҚ°зҡ„пјҢеҲҷе…ҲжҠҠе·Ұиҫ№зҡ„йғҪдҫқж¬ЎеҺӢж ҲпјҢз„¶еҗҺеҺӢе…ҘжңҖеҗҺдёҖдёӘпјҢејҖе§ӢйҖҗжӯҘеј№еҮәпјҢеј№еҮәе°ұжү“еҚ°дёҖдёӘпјҢз„¶еҗҺйҒҚеҺҶеј№еҮәзҡ„еҸіеӯҗж ‘пјҢд№ҹжҳҜдёҖж ·зҡ„ж–№жі•пјҢдҫқж¬ЎеҫӘзҺҜпјҢзӣҙеҲ°е…ЁйғЁйҖҖеҮәпјҢжү“еҚ°еҮәжңҖйЎ¶еұӮзҡ„rootгҖӮ

еҗҺеәҸйҒҚеҺҶ(ж·ұеәҰдјҳе…Ҳ)

* жҢҮе…Ҳи®ҝй—®еӯҗж ‘пјҢз„¶еҗҺи®ҝй—®ж №зҡ„йҒҚеҺҶж–№ејҸгҖӮеӯҗж ‘е…Ҳе·ҰеҗҺеҸіеңЁж №иҠӮзӮ№гҖӮ * йЎәеәҸ `A, C, E, D, B, H, I, G, F.`

е Ҷж Ҳд»Јз Ғе®һзҺ°пјҡ

и§ЈйҮҠпјҡеӣ дёәйЎәеәҸдёәд»ҺжңҖе·ҰиҠӮзӮ№ејҖе§ӢпјҢдё”йңҖиҰҒеҲӨж–ӯжҳҜеҗҰеӯҳеңЁж №зҡ„е·ҰеҸіеӯҗиҠӮзӮ№пјҢдё”дёҚжҳҜиҝһз»ӯпјҢжүҖд»ҘйңҖиҰҒз”ЁеҲ°дёӨдёӘж ҲгҖӮmystack1ж”ҫе…Ҙзҡ„жҳҜеҪ“еүҚйҒҚеҺҶзҡ„иҠӮзӮ№пјҢз„¶еҗҺеҫҖmystack2дёӯеҺӢе…ҘпјҢиҝҷдёӘиЎҢдёәд№ӢеүҚиҝҳеҜ№иҠӮзӮ№иҝӣиЎҢе·ҰеҸіеҲӨж–ӯпјҢе…ҲеҫҖstack1дёӯеҺӢе…Ҙе·Ұиҫ№зҡ„иҠӮзӮ№пјҢз„¶еҗҺеҺӢе…ҘеҸіиҫ№зҡ„иҠӮзӮ№пјҢиҪ®еҲ°дёӢж¬ЎеҫӘзҺҜзҡ„ж—¶еҖҷпјҢеҗҺж”ҫе…Ҙstack1дёӯзҡ„(еҸіеӯҗж ‘)иў«е…ҲpopеҮәжқҘпјҢз„¶еҗҺиў«жҸ’е…ҘеҲ°stack2дёӯпјҢжӯӨж—¶stack2дёӯеӯҳж”ҫзҡ„йЎәеәҸпјҢж №иҠӮзӮ№пјҢз„¶еҗҺеҸіиҠӮзӮ№пјҢжҺҘзқҖе·ҰиҠӮзӮ№пјҢдҫқж¬ЎеҫӘзҺҜпјҢзӣҙеҲ°жҠҠstack2з»ҷеҺӢж»ЎгҖӮжңҖеҗҺдёҖдёӘwhileжү§иЎҢзҡ„ж—¶еҖҷпјҢжҢүз…§е·ҰеҸіж №зҡ„йЎәеәҸжү“еҚ°еҮәжқҘпјҢе°ұе®һзҺ°дәҶгҖӮ

еұӮзә§йҒҚеҺҶ(е№ҝеәҰдјҳе…Ҳ)

* е№ҝеәҰдјҳе…ҲйҒҚеҺҶдјҡе…Ҳи®ҝй—®зҰ»ж №иҠӮзӮ№жңҖиҝ‘зҡ„иҠӮзӮ№гҖӮдәҢеҸүж ‘зҡ„е№ҝеәҰдјҳе…ҲйҒҚеҺҶеҸҲз§°жҢүеұӮж¬ЎйҒҚеҺҶгҖӮз®—жі•еҖҹеҠ©йҳҹеҲ—е®һзҺ°гҖӮ * йЎәеәҸ `F, B, G, A, D, I, C, E, H.`

йҳҹеҲ—д»Јз Ғе®һзҺ°пјҡ

и§ЈйҮҠпјҡйҖҡиҝҮйҳҹеҲ—ж–№ејҸе®һзҺ°пјҢжҜ”иҫғз®ҖеҚ•пјҢжҠ“еҲ°е…ғзҙ appendиҝӣеҺ»пјҢдёҖе·ҰдёҖеҸіж–№ејҸж”ҫе…ҘпјҢеј№еҮәзҡ„ж—¶еҖҷеј№еҮәз¬¬дёҖдёӘе…ғзҙ пјҢе№¶дё”жү“еҚ°гҖӮ
