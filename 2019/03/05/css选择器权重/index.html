CSS选择器权重

Posted on 2019-03-05 Edited on 2020-12-19

CSS选择器类型

  • id选择器
    #id
  • class选择器
    .class
  • *选择器
    \*
  • 元素选择器
    element
  • 伪类
    :hover
  • 父子选择器
    element element
    element>element
  • 属性选择器
    [attribute=value]
  • 组选择器
    element,element1,element2

CSS选择器优先级

优先级高的覆盖优先级低的样式。
!important > 行间样式(直接元素内写style=xxx的方式) > id > class | 属性 > 标签选择器 > 通配符
其规则其实是由选择器权重实现的。

CSS选择器权重

权重大的覆盖权重小的，不同的选择器有不同的权重。

  • !important Infinity
  • 行间样式 1000
  • id 100
  • class | 属性 | 伪类 10
  • 标签 | 伪元素 1
  • 通配符 0

进制据说是265。
